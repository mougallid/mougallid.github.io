Versionner en local

  • git init:Initialiser le depôt (se mettre sur le bon dossier), mieux a faire depuis Github.com
  • git add . : ajouter tous le modification (le . symbolise tout)
  • git commit -m "explication" : Créer un nouveau commit. git add posse le fichier en zonne d'index. git commit les sauvegarde réellement dans un nouveau commit.
